# Lesson 10: Aggregation (COUNT, SUM, AVG, GROUP BY)

## 📌 Overview

This project demonstrates the use of SQL aggregation functions to analyze and summarize data. It focuses on key functions such as `COUNT`, `SUM`, and `AVG`, along with the `GROUP BY` clause to organize results into meaningful categories.

## 🛠 Tools Used

* Visual Studio Code
* SQL Server Management Studio (SSMS)
* Microsoft SQL Server

## 📊 Key Concepts Covered

* `COUNT()` – Counts the number of records
* `SUM()` – Calculates the total value of a column
* `AVG()` – Computes the average value
* `GROUP BY` – Groups rows with similar values for aggregation

## 📂 Project Structure

* SQL scripts demonstrating each aggregation function
* Example queries using `GROUP BY`
* Sample datasets for testing and practice

## 🚀 How to Run

1. Open the project in Visual Studio Code or SSMS
2. Connect to your SQL Server instance
3. Run the SQL scripts step by step
4. Review the output results

## 📈 Purpose

The goal of this project is to build a solid understanding of SQL data aggregation techniques, which are essential for data analysis and reporting.
